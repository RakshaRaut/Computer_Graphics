Computer Graphics Labs Repository

This repository contains my lab work for Computer Graphics, featuring code written in C using the `<X11/Xlib.h>` library and OpenGL on Linux. To run these programs on your system, you can follow the setup instructions provided within the repository to install the necessary libraries.

1.Install X11 Development Libraries:
   ```bash
   sudo apt-get install libx11-dev
   ```

2.Install OpenGL Development Libraries:
   ```bash
   sudo apt-get install libgl1-mesa-dev
   ```

3.Install Additional OpenGL Utility Libraries (Optional):
   ```bash
   sudo apt-get install freeglut3-dev
   ```
